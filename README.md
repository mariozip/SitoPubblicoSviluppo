# Se volessi completare il gioco indovinello delle località:
Salva le immagini dei luoghi all'interno di questa nuova cartella. I nomi dei file devono corrispondere esattamente a quelli presenti nel codice JavaScript (js/GuessThePlace.js):
colosseo.jpg
tour-eiffel.jpg
statua-liberta.jpg
piramidi-giza.jpg
sydney-opera-house.jpg
machu-picchu.jpg
Puoi usare immagini che trovi online, ma assicurati che siano di pubblico dominio o con una licenza che ne permetta l'uso.

Una volta aggiunte le immagini nella posizione corretta, ricarica la pagina e vedrai che il gioco funzionerà come previsto! Scusa ancora per la confusione
